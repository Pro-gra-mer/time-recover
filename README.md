# Time Recover - Landing Page Ultra-Premium

Landing page de alto impacto para servicios de automatización con IA orientada a empresarios de alto rendimiento. Diseñada para convertir leads de alto valor mediante una estética exclusiva y minimalista.

## Stack Tecnológico

- **HTML5 & Vanilla JS**: Estructura sólida y micro-interacciones.
- **Tailwind CSS**: Base de estilos moderna y responsive.
- **Custom CSS (Glassmorphism)**: Efectos de transparencia y desenfoque para un look premium.
- **Google Fonts**: Tipografías 'Outfit' y 'Inter' para máxima legibilidad y estilo.
- **Formspree**: Gestión de leads sin backend.

## Características Renovadas

- **Estética "SaaS Premium"**: Tema oscuro con gradientes mesh y efectos de cristal (glassmorphism).
- **Identidad Visual**: Logo personalizado y visuales abstractos de alta fidelidad.
- **Micro-interacciones**: Navegación dinámica, scroll reveal y efectos hover sofisticados.
- **Optimizado para Conversión**: Embudo de ventas narrativo que guía al usuario desde el problema a la inversión.
- **Responsive Design**: Adaptación impecable a móviles, tablets y escritorio.

## Configuración y Despliegue

1. **Formspree**: Abre `index.html` y reemplaza todas las instancias de `tu-form-id` con tu ID real de Formspree para recibir los datos del formulario.
2. **Assets**: Asegúrate de que los archivos `time_recover_logo_...png` y `ai_abstract_hero_...png` estén en la misma carpeta que el `index.html`.
3. **Hosting**: Sube los archivos a cualquier hosting estático (Vercel, Netlify, GitHub Pages, etc.).

## Estructura del Proyecto

- `index.html`: La experiencia completa de la landing page.
- `*.png`: Logos y recursos visuales generados específicamente para la marca.
- `README.md`: Este manual de uso.
